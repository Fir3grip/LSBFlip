# LSBFlip
The is the GitHub repo for CSEC 759. The project that was created for this class is a tool called LSBFlip. 
It blocks canvas fingerprinting by flipping the LSBs in the generated HTML5 canvas

canitrack_test contains the html file and test harness that were talked about during the presentation.

canvas-bitflip-extension contains the failed extension that we tried to create out of LSBFlip.

Presentation_materials contains all of the materials that were used for the in class presentation.
This includes the slides as well as a video of the presentation.
